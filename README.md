# How to install the code

```
npm install
```
open a new terminal
```
truffle develop
copy the nemonic 
logout from the metamask and Import using account seed phrase
copy paste the nemonic
migrate --reset in the same terminal and keep that terminal open
```

open a new terminal
``` 
cd app or client
npm install
npm start
```

enjoy the code happy coding
